# Privacy-Policy-ART-OF-LOVE-Bot
ART OF LOVE is a Discord bot designed to provide moderation, role management, and automation features.
Data Collected:
The bot may process the following data within Discord servers:
User IDs
Server IDs
Role IDs
Message content (processed temporarily for moderation and commands)
How Data is Used:
To detect spam, links, and rule violations
To execute commands and automate server features
To manage roles, mutes, and moderation systems
Data Storage:
No message content is stored permanently
Only necessary configuration data (such as role IDs and server settings) may be stored
Data Sharing:
No user data is sold or shared with third parties
User Control:
Users can avoid interaction by not using bot commands
Server administrators control bot usage within their servers
Security:
Reasonable measures are taken to protect stored data
Changes:
This policy may be updated at any time
