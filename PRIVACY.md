# Privacy Policy – LOVE OF ART Bot

Last Updated: 2026

## Overview
LOVE OF ART is a Discord bot designed to provide moderation, automation, and server management features. This Privacy Policy explains what data is processed and how it is used.

---

## Data Collected

The bot may process the following data within Discord servers:

- User IDs
- Server (Guild) IDs
- Role IDs
- Channel IDs
- Message content (processed temporarily)

---

## How Data is Used

Data is used strictly for bot functionality, including:

- Detecting spam, links, and rule violations
- Executing commands and interactions
- Managing roles, mutes, and moderation systems
- Logging server activity (joins, leaves, commands)

---

## Data Storage

- Message content is **not stored permanently**
- Configuration data (such as server settings, role IDs, and moderation data) may be stored locally to ensure proper bot functionality
- Stored data is limited to what is necessary for operation

---

## Data Sharing

- No data is sold, shared, or distributed to third parties
- Data is only used within the bot’s systems

---

## User Control

- Users can avoid interaction by not using bot commands
- Server administrators control how the bot is used within their servers

---

## Data Security

Reasonable measures are taken to protect stored data. However, no system can guarantee absolute security.

---

## Changes to This Policy

This Privacy Policy may be updated at any time. Continued use of the bot implies acceptance of any changes.

---

## Contact

For questions or concerns, contact the bot developer through Discord.
